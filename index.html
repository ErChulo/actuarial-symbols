üìö Component Reference

1. Generic Symbol <act-symbol>

The most flexible component. Implements the full 4-corner notation.

Attribute Description Example
symbol Main symbol (required) "A", "a", "E", "p", "q"
lr Lower-right subscript (required) "x", "xy", "x:n"
ll Lower-left subscript "n", "k", "t"
ul Upper-left superscript "1", "2"
ur Upper-right superscript "(m)", "{1}"
p Premium/Reserve wrapper "P", "V", "W"
decoration Symbol decoration "bar", "ddot", "ring"
precedence JSON array for precedence numbers [{pos:0,num:1}]
last-survivor Apply overline (boolean) - 
<act-symbol symbol="A" lr="x"></act-symbol>
<act-symbol ll="n" ul="2" symbol="A" lr="x" ur="(m)"></act-symbol>
<act-symbol symbol="A" lr="xyz" precedence='[{"pos":2,"num":2}]'></act-symbol>
